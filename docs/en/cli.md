# CLI Reference

## run

```bash
pyscf_auto run input.xyz --config run_config.yaml
```

Key options:

- `--background`: enqueue a background run
- `--profile`: record SCF/gradient/Hessian timing
- `--resume <RUN_DIR>`: resume an existing run
- `--force-resume`: allow resume from completed/failed states
- `--queue-priority <N>`: queue priority
- `--queue-max-runtime <SEC>`: queue timeout
- `--scan-dimension ...`, `--scan-mode ...`, `--scan-grid ...`: CLI scan overrides

## status

```bash
pyscf_auto status

pyscf_auto status --recent 5
```

## queue

```bash
pyscf_auto queue status
pyscf_auto queue cancel <RUN_ID>
pyscf_auto queue retry <RUN_ID>
pyscf_auto queue requeue-failed
pyscf_auto queue prune --keep-days 30
pyscf_auto queue archive
```

## validate-config

```bash
pyscf_auto validate-config run_config.yaml
```

## doctor

```bash
pyscf_auto doctor
```

## smoke-test

```bash
pyscf_auto smoke-test --smoke-mode quick
```

## scan-point (advanced)

Use a manifest generated by `scan.executor: manifest`.

```bash
pyscf_auto scan-point --manifest path/to/scan_manifest.json --index 0
```
